# AI Study Assistant

An AI-powered chatbot that helps students understand their notes using Retrieval-Augmented Generation (RAG).

## Features
- Upload notes and ask questions
- Concise and detailed response modes
- Web search fallback
- Question generation

## Tech Stack
- Streamlit
- FAISS
- Sentence Transformers
- Groq LLM

## How to Run
pip install -r requirements.txt
streamlit run app.py
